## **Snake Game in Python**

A simple implementation of the classic Snake game in Python. The game is played on a rectangular grid of size 15x17, where the player controls a snake that grows longer each time it eats an apple. The game ends if the snake runs into a wall or itself.

### **Features:**

Grid size: 15 rows x 17 columns

Snake starts with 3 cells

Apples appear randomly on empty cells

Eating an apple grows the snake

Game ends when the snake hits a wall or its own body

Keyboard controls for moving the snake
